Description about the following news

The new Airtel Rs 401 plan has its advantages and disadvantages too. On one hand, it might attract the customers with a free subscription to a streaming app while on the other hand, it can put off customers for not providing any calling or SMS benefits. Yes, you read that right. The Rs 401 data pack doesn’t provide voice calling and SMS benefits. The plan only provides 3GB data per day for 28 days. However, the free subscription that it provides is valid for 365 days.
